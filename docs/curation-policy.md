# Curation and release policy

Catalog facts are traced to a dataset paper, official project page, or archival
repository. Aggregator pages may help discovery but are not treated as final
evidence. Ambiguous fields are represented as `null` or `unknown`, not guessed.

Each entry tracks three different concepts:

- **Discoverable**: a stable landing page exists.
- **Downloadable**: data can currently be obtained under stated conditions.
- **Redistributable**: the license permits WiSenseHub to host a copy or derivative.

The hub prefers official downloads. Mirrors are labeled and never silently
substituted. Restricted releases receive metadata, instructions, and local
conversion code but no republished bytes.
