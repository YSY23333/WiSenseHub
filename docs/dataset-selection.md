# Initial dataset selection

The first release balances task coverage, hardware diversity, openness, and
standardization feasibility. It includes the datasets compared by CSI-Bench and
task-specific extensions for continuous TAD/TAL, respiration, occupancy,
through-wall sensing, and multimodal sensing.

WiFiTAD and XRF V2 form a dedicated continuous activity category. XRF55 remains
a clip-level HAR dataset; XRF V2 is a distinct release for TAL and action
summarization.
